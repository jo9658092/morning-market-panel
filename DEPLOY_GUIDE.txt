Morning Market Panel - Free Deployment Guide

1) Create a GitHub repository and upload all files.
2) In GitHub Settings > Pages, deploy from the main branch root or use GitHub Actions Pages workflow if preferred.
3) In GitHub repository Settings > Secrets and variables > Actions, add:
   - ALPHAVANTAGE_API_KEY = your free API key
4) The workflow file `.github/workflows/update-market-snapshot.yml` runs twice daily.
   - 22:00 UTC = 07:00 KST
   - 06:00 UTC = 15:00 KST
5) The script `scripts/update_data.py` writes `data/latest.json`.
6) The app fetches `./data/latest.json` on load and shows the latest snapshot.

Important
- Free data is a substitute panel, not Bloomberg original data.
- GitHub Actions cron can run on schedule, but execution timing may not be exact to the minute.
- GitHub Pages can host the PWA for free.
